Detailed Description

template<typename T>
class core_lib::threads::ConcurrentQueue< T >

Class defining a concurrent queue.

This class implements a fully thread-safe queue, that can be used as single/multiple producer and single/multiple consumer, in any combination.

template<typename T>
void core_lib::threads::ConcurrentQueue< T >::Clear ( )
inline

Clear the queue.

This method will clear the queue of its contents and if the correct options are set then the items on the queue are deleted.

The method should only be called when no threads are blocked on any of the pop methods.
