Detailed Description

Thread base class.

This abstract class can be used as a base class for objects that need to be threaded. It neatly wraps all the useful functionality of std::thread in a usable way.

void core_lib::threads::ThreadBase::ProcessTerminationConditions ( )
protectedvirtual

Perform any special termination actions.

This function performs no actions in the base class definition but can be overridden in the derived class to perform any special termination actions that are required after the terminating flag is set but before we call join on our underlying std::thread object.

Reimplemented in core_lib::threads::EventThread, core_lib::threads::MessageQueueThread< MessageId, MessageType, MapType >, core_lib::threads::MessageQueueThread< int, dl_private::LogQueueMessage >, and core_lib::threads::ThreadRunner.
