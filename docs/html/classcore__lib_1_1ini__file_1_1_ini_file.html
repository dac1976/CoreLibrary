Detailed Description

Ini file class.

Here is an example of what an ini file could look like:

; I am an opening comment.

[Section1] ; I am a comment in a section. Section1_Key1=Section1_Value1 Section1_Key2=Section1_Value2 Section1_Key3 = Section1_Value3

[Section2] Section2_Key1=Section2_Value1 Section2_Key2 = Section2_Value2 ; I am also a comment in a section. Section2_Key3=Section2_Value3 ; I am yet another comment in a section.

Please note that the supported comment delimiter is ';'. Also note that blank lines and comments are preserved when loading an ini file. However, leading and trailing whitespace in section, key or value items are removed.
