Detailed Description

template<class Formatter>
class core_lib::log::DebugLog< Formatter >

DebugLog class.

Templated class to perform logging. The class is threaded and thread safe.

The template args comprise the a formatter type and which should be a function object with same prototype as DefaultLogFormat::operator ()(). The second arg is optional and controls the size at which the log will close and switch to a new file. Only 2 files ever exist the <log>.txt and <log>_old.txt. The default log size if 5MiB.

You can dynamically filter out log message from appearing in the log file based on adding message levels to the filter set.
