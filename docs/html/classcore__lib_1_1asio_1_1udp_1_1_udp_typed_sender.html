Detailed Description

template<typename MsgBldr>
class core_lib::asio::udp::UdpTypedSender< MsgBldr >

A generic UDP sender.

The template argument defines a message builder object that must have an interface compatible with that of the class asio::messages::MessageBuilder.

This class forms the underpinnings of the SimpleUdpSender class.

This is also the class to be used when the user wants to specify their own message builder type and message header type.

Typically use this constructor when managing a bool of threads using an instance of IoContextThreadGroup in your application to manage a pool of std::threads.
