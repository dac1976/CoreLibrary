Split a string into two parts given delimiters.

Parameters
[out]subStr1- First output substring.
[out]subStr2- Second output substring.
[in]toSplit- Input string to be split into two.
[in]delim- Delimiters to look for to split around.
[in]option- Options to formt resultant substrings.

Given an input string and a string containing delimiters the input string is split into two parts either side of the deilimiter string. If none of the delimiters can be found then a xSplitStringBadDelim exception is thrown. If the string will be split into more than two substrings then a xSplitStringTooManySubStrings exception is thrown.
