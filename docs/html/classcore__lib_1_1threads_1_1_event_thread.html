EventThread constructor.

Parameters
[in]eventCallback- Function object to be called when event ticks.
[in]eventPeriod- Period between event being triggered.
[in]delayedStart- When true user must called Start() manually.
[in]timeUnit- The unit of time associated with the period.

If eventPeriod is 0 then the thread uses a blocking Wait() instead of a WaitForTime. In this case you must trigger the event to wait up from outside this object using ForceTick().
